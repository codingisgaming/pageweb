 Description
 Une Page  web interactive de coaching sportif pour recommander des exercices personnalisés basés sur
les réponses de l'utilisateur à un quiz. L'application propose trois catégories 
d'exercices : Musculation, Pilates et Yoga, avec des démonstrations vidéo et des instructions détaillées.


Lancement local
Télécharger tous les fichiers dans un même dossier
Ouvrir index.html dans un navigateur web moderne
Vérifier que le dossier resources/ contient toutes les images et vidéos nécessaires
Autoriser l'exécution des scripts si le navigateur le demande
Cliquer sur "Parler à Obélix" pour commencer le quiz


Prérequis
Navigateur web moderne (Chrome 90+, Firefox 88+, Safari 14+)

Connexion internet pour les polices Google Fonts

JavaScript activé

Espace disque pour les médias (environ 50-100 MB selon les vidéos)


Comment utiliser l'application
Page d'accueil : Cliquez sur le bouton "Parler à Obélix"

Quiz interactif : Répondez aux  questions pour déterminer votre profil sportif

Résultats : Recevez votre recommandation personnalisée (Musculation, Pilates ou Yoga)

Exercices : Consultez les exercices recommandés avec démonstrations

Recommencer : Utilisez le bouton pour refaire le quiz


Architecture
en utlisant des langages natives pour inclure le theme : 
HTML5 : Structure sémantique avec balises modernes

CSS3 : Variables CSS (:root), Grid, Flexbox, animations

JavaScript Vanilla : Pas de frameworks externes

Modèle objet : Organisation en fonctions modulaires

Design et thème : 
Thème "village gaulois" : Opposé à l'esthétique "big tech" minimaliste
Palette de couleurs chaleureuses : Or, brun, parchemin

Polices médiévales : Cinzel et MedievalSharp pour l'immersion

Effets visuels : Ombres portées, lueurs, animations subtiles

Design immersif : Interface qui raconte une histoire
